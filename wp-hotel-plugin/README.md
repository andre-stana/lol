# WP Hotel Plugin

Ce plugin WordPress permet aux utilisateurs de créer des produits WooCommerce dans la catégorie "hôtel". Il fournit un formulaire sécurisé pour ajouter des informations sur les hôtels, accessible uniquement aux utilisateurs ayant le rôle `um_hotels` ou aux administrateurs.

## Fonctionnalités

- Création de produits WooCommerce pour les hôtels.
- Champs pour le nom de l'hôtel, photos, description, dates de disponibilité et adresse.
- Accès restreint au formulaire pour les utilisateurs autorisés.

## Installation

1. Téléchargez le plugin.
2. Décompressez le fichier zip.
3. Téléversez le dossier `wp-hotel-plugin` dans le répertoire `/wp-content/plugins/`.
4. Activez le plugin via le menu `Plugins` dans WordPress.

## Utilisation

- Après activation, seuls les utilisateurs avec le rôle `um_hotels` ou les administrateurs peuvent accéder au formulaire de création d'hôtel.
- Remplissez le formulaire avec les informations requises et soumettez-le pour créer un nouveau produit hôtelier.

## Désinstallation

Pour désinstaller le plugin, allez dans le menu `Plugins`, désactivez le plugin, puis supprimez-le. Toutes les données associées seront supprimées.

## Sécurité

Le plugin est conçu pour être sécurisé, avec des vérifications de permissions et une validation des données pour protéger la base de données.
