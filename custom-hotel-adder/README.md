# Custom Hotel Adder

## Description
Le plugin **Custom Hotel Adder** permet d'ajouter dynamiquement des hôtels à votre site WooCommerce. Il offre une gestion complète des labels depuis l'administration et inclut des fonctionnalités avancées pour la gestion des images, y compris un carrousel pour afficher les photos des hôtels.

## Installation

1. Téléchargez le plugin et décompressez le fichier ZIP.
2. Téléversez le dossier `custom-hotel-adder` dans le répertoire `/wp-content/plugins/` de votre installation WordPress.
3. Activez le plugin via le menu `Plugins` dans WordPress.

## Utilisation

- Une fois le plugin activé, vous verrez un nouveau menu intitulé **Formulaire Hôtels** dans le tableau de bord WordPress.
- Cliquez sur ce menu pour accéder à la page de configuration où vous pouvez personnaliser les labels du formulaire.
- Utilisez le shortcode `[add_hotel_form]` pour afficher le formulaire d'ajout d'hôtel sur n'importe quelle page ou publication.

## Fonctionnalités

- **Ajout d'Hôtels** : Permet aux utilisateurs de soumettre des hôtels avec des informations personnalisées.
- **Gestion des Images** : Inclut un carrousel pour visualiser les images des hôtels et la possibilité de supprimer des images sélectionnées.
- **Personnalisation des Labels** : Les administrateurs peuvent modifier les labels des champs du formulaire via l'interface d'administration.

## Dépendances

- WooCommerce : Ce plugin nécessite WooCommerce pour fonctionner correctement.
- WordPress : Assurez-vous d'utiliser une version récente de WordPress.

## Configuration

Aucune configuration supplémentaire n'est nécessaire après l'installation. Vous pouvez commencer à ajouter des hôtels immédiatement après avoir configuré les labels.
